Fun Scrabble

Welcome to "Fun Scrabble" game!!! A 2 player verses word game!!!

The game is very easy to play and unlike scrabble a word will be give, and you only have to make up a different word by rearranging the letters of the word.
Don't Worry!!! Options will be given, you only have to Select the correct Word. Player with the Highest score wins!!

Conditions 1: A host computer to start the game.
Conditions 2: Two players need to be joined.
Conditions 3: Need to be in a local connection.
Best Experience, using a computer as host and two mobile phones/tabs/PC for playing
